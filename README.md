# Управленка Dashboard

Dashboard для торговой компании (замороженные ягоды): пульс, план продаж, сделки, деньги, ROI, дебиторка, склад.

## Запуск (локально)

```bash
cd upravlenka-dashboard
npm run dev
```

Откройте адрес из терминала (http://localhost:5173 или 5174).

> Зависимости уже в репозитории (`node_modules/`). `npm install` нужен только если папки нет.

**Развёртывание для друга:** см. [RAZVERTYVANIE.md](./RAZVERTYVANIE.md)

## Экраны

| Вкладка | Содержание |
|---------|------------|
| **Пульс** | Рентабельность вчера, дыра по деньгам, ROI, умные выводы |
| **План продаж** | План vs факт: выручка, кг, маржа |
| **Сделки** | История + незавершённые, фильтры по периоду |
| **Деньги** | Счета, обязательства, график поступлений |
| **ROI** | ROI по типам капитала, оборачиваемость |
| **Дебиторка** | Просрочка, график оплат |
| **Склад** | Остатки ягод по группам |
| **Закупки** | Товар в пути, кредиторка |
| **Месяц** | Оперсрез, опербаланс |

Данные: Excel «Управленка» + импровизированные блоки (счета, ROI, история сделок по дням).

```bash
npm run extract   # обновить из Excel
```
